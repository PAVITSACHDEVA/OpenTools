Image Generator & Editor

Features:
- Random images from Lorem Picsum
- Upload your own images
- Real-time filters
- High quality downloads
- Keyboard shortcuts

Run:
Just open index.html in a browser.

Shortcuts:
G = Generate
R = Reset
O = Open image

Extend:
Add presets in script.js
Add more filters in defs array
Add effects in applyFilters()

No dependencies needed.
